## 介绍
**mypages** 是一个致力于聚合多社交平台用户主页的小型单页应用。
主要为平台管理、平台观点管理、关注用户管理、用户类型管理、用户标签管理、检查用户内容更新、同步用户头像昵称等业务功能提供 restful 接口支持。
